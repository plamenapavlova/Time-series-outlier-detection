# Student-performance-analysis

# Global Math Performance Analysis: Gender and Regional Comparisons

## Project Overview

This project explores **global mathematics performance differences between boys and girls** across continents and over time. The analysis focuses on identifying patterns in average scores, trends in Europe, and comparisons between Europe and Asia.

The project includes:

1. Gender comparison across continents
2. Trends in mathematics scores over time
3. Weighted and unweighted averages in Europe
4. Regional comparison between Europe and Asia

The goal is to use **exploratory data analysis and visualization** to highlight gender gaps and regional performance differences.

## Methods

The analysis was conducted using:

* Exploratory Data Analysis (EDA)
* Data aggregation and averaging
* Weighted and unweighted statistical comparisons
* Data visualization

---

## Conclusion

The analysis reveals several key insights:

1. Boys tend to score slightly higher than girls in mathematics across most regions.
2. Europe generally shows higher average scores compared to Asia.
3. Both weighted and unweighted analyses indicate a **declining trend in European mathematics performance in recent years**.
4. The gender performance gap remains relatively stable over time.
